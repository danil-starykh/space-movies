# Space Movies
Best application to find your favorite movie

### Used technologies:

- React
- Redux
- TypeScript
- SaaS

### Implemented pages:

- List of movies
- Movie details
- Error Page ( Error Bundler )
- Not Found Page ( Wrong url )
